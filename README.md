# 照片转完美像素画工具

一个简单易用的工具，将任意照片转换为高质量的像素画，并使用PerfectPixel技术进行优化。

## 功能特点

- **傻瓜式操作**：基于Gradio的用户界面，一键转换
- **自动网格检测**：智能检测最佳网格大小
- **边缘对齐**：确保网格与图像内容完美匹配
- **多种采样方法**：支持中心、中值、多数三种采样方式
- **参数可调**：可以调整像素大小、颜色数量和像素密度
- **高质量输出**：生成的像素画适合用于游戏开发、艺术创作等
- **实时摄像头**：支持实时摄像头输入进行像素化处理

## 技术依赖

本项目使用了 [perfectPixel](https://github.com/theamusing/perfectPixel) 项目的核心算法来优化像素画效果。在此特别感谢 perfectPixel 项目的作者。

## 安装步骤

### 1. 克隆项目

```bash
git clone <项目地址>
cd 照片转完美像素画
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

依赖包包括：
- numpy
- opencv-python
- Pillow
- gradio

## 使用方法

### 启动用户界面

```bash
python ui/workflow_app.py
```

然后在浏览器中打开 http://localhost:7896，即可使用图形界面上传照片并生成像素画。

## 功能说明

1. **图片处理**：
   - 上传图片：点击中间的"上传图片"区域，选择要处理的图片
   - 调整参数：在左侧调整像素大小、颜色数量等参数
   - 生成像素画：点击"生成像素画"按钮，等待处理完成
   - 查看结果：在右侧查看像素画效果和PerfectPixel优化效果

2. **实时摄像头**：
   - 打开摄像头：点击"📷 打开摄像头"按钮（会请求摄像头权限）
   - 开始转换：点击"▶️ 开始转换"按钮开始实时处理
   - 调整参数：在左侧调整参数查看实时效果
   - 停止转换：点击"⏸️ 停止转换"按钮暂停处理
   - 关闭摄像头：点击"🔴 关闭摄像头"按钮完全关闭摄像头

## 参数说明

| 参数 | 说明 | 默认值 | 范围 |
|------|------|--------|------|
| 像素大小 | 控制像素块的大小，值越大，像素块越大 | 8 | 4-32 |
| 颜色数量 | 控制颜色的数量，值越小，颜色越少 | 8 | 2-32 |
| 像素数量 | 控制PerfectPixel优化后的像素密度，值越大越精细 | 20 | 5-100 |
| 采样方法 | 选择网格采样的方式 | majority | center, median, majority |
| 放大倍数 | 控制输出图片的放大倍数 | 4 | 1-16 |
| 添加网格线 | 是否在输出图片上添加网格线 | False | True/False |

## 工作原理

1. **风格转换**：将普通照片转换为像素风格
2. **网格检测**：自动检测最佳网格大小（使用perfectPixel算法）
3. **网格优化**：将网格对齐到图像边缘（使用perfectPixel算法）
4. **网格采样**：根据优化后的网格对图像进行采样

## 算法说明

### 为什么使用这种算法

1. **传统像素化方法的局限性**：传统的像素化方法只是简单地缩小图像再放大，缺乏对图像结构的理解，导致生成的像素画效果生硬，边缘不清晰。

2. **PerfectPixel算法的优势**：
   - **智能网格检测**：使用FFT（快速傅里叶变换）分析图像的频率特征，自动检测最佳网格大小
   - **边缘对齐**：使用Sobel边缘检测算法，将网格线与图像边缘对齐，使像素画更加自然
   - **多种采样方法**：支持中心采样、中值采样和多数采样，适应不同类型的图像
   - **高质量输出**：生成的像素画边缘清晰，色彩过渡自然，适合用于游戏开发和艺术创作

### 算法工作流程

1. **输入处理**：读取并预处理输入图像，确保格式正确
2. **像素化**：将图像缩小后再放大，使用最近邻插值产生初步的像素化效果
3. **色彩量化**：使用K-means聚类减少图像颜色数量，实现像素风格
4. **网格检测**：使用FFT分析图像频率，自动检测最佳网格大小
5. **网格优化**：根据边缘检测结果，调整网格线位置，使其与图像边缘对齐
6. **网格采样**：根据选择的采样方法（中心、中值或多数）对图像进行采样
7. **后处理**：添加网格线（可选）并放大输出结果

### 采样方法说明

- **中心采样**：取每个网格中心的像素值，计算速度快，适合线条清晰的图像
- **中值采样**：取每个网格的中值像素值，能有效去除噪声，适合复杂纹理的图像
- **多数采样**：取每个网格中最常见的像素值，能保持图像的整体风格，适合色彩丰富的图像

## 注意事项

- 输入照片建议使用清晰、对比度高的图像
- 对于复杂场景，可能需要调整参数以获得最佳效果
- 生成的像素画可以根据需要进一步编辑和处理
- 实时摄像头功能需要浏览器权限

## 隐私说明

- **摄像头使用**：实时摄像头功能仅在用户明确授权后使用，不会在未经许可的情况下访问摄像头
- **数据处理**：所有图像处理均在本地进行，不会将用户的图像数据上传到任何服务器
- **数据存储**：实时摄像头处理过程不会自动保存任何视频或图像数据，除非用户手动下载输出结果
- **权限管理**：用户可以随时在浏览器设置中撤销摄像头权限
